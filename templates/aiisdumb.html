There Was An Error Processing Your Log In Page

Our system uses AI to log into different assignment services, but sometimes it messes up. Help it out by showing it how to interact with your organization's sign-in page

Please select all the relevant elements on your page (text boxes and buttons) by drawing a box over them with your finger. It is important that you select all the buttons on the page, even if you would not normally click on them when signing in. We just need to know that they are there. However, you should not select links, only buttons. After selecting each element, use the dropdown menu next to it to select what you want that element to do. Select "None" if you would not normally interact with the element when signing in to your account.
